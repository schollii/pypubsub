Upgrade notes
^^^^^^^^^^^^^

- Version now comes from git tags; bump by tagging (e.g., `v4.0.7`), no manual `__version__` edits.
- If you need wx examples, install wxPython 4.2.4 via the extras feed and SDL2 runtime on Ubuntu (see README).
- Python <3.7 is no longer supported/tested.
